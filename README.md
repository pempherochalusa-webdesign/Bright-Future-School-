# School Management Web App

This is a simple School Management System built with HTML, CSS and JavaScript.

## Features
- Student registration
- Login system (Admin, Teacher, Student)
- Teacher can enter marks
- Student can view results
- Ranking system
- Secure pages using localStorage

## Files
- index.html – Home page
- login.html – Login page
- register.html – Student registration
- teacher.html – Teacher dashboard
- student.html – Student dashboard
- admin.html – Admin dashboard
- script.js – JavaScript logic
- style.css – Styling

## How to use
1. Open login.html
2. Login as:
   - Admin (admin / admin123)
   - Teacher (teacher / teacher123)
   - Or register as a student
3. Enter marks and view results
